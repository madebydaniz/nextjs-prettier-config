# NextJs Prettier Config

> NextJs [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ npm install -D git+ssh://git@github.com:madebydaniz/nextjs-prettier-config.git
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "nextjs-prettier-config",
}
```
